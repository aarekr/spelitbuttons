# Spelit Buttons Game

### Game
Four circles randomly appear on the screen and player has to tap them. There are three levels players can choose from. Difficulty level affects how many points players have to get and for how long the game lasts. Target audience is children, so tapping circles in other than appearence order will not end the game or cause point losses. The game remembers the number of times player has passed each level and the times time has ran out.
